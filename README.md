# Анализ удовлетворенности клиентов. Исследование комфорта при работе климатического оборудования.


### Описание проекта

В проекте проведён анализ удовлетворенности пользователей условиями микроклимата в офисных помещениях. Исследование основано на данных о работе климатического оборудования и субъективных оценках комфорта респондентов в разных странах и климатических условиях.

**Цель** — определить, какие факторы влияют на комфорт и в какой тип климатического оборудования целесообразно инвестировать в зависимости от региона.

**Бизнес-задача** -- определить:
- какой тип систем охлаждения обеспечивает более высокий уровень комфорта
- какие параметры среды наиболее значимы для пользователей
- региональные особенности восприятия микроклимата


### Данные

Датасет включает:
- параметры окружающей среды (температура, влажность, скорость воздуха)
- типы охлаждения и отопления
- характеристики помещений
- демографические данные респондентов
- оценки комфорта

### Ход работы

**Предобработка**
- приведение названий столбцов к стандарту PEP8
- проверка и корректировка типов данных
- обработка пропусков
- выявление и обработка выбросов
- удаление дубликатов

**Анализ данных**
- исследование распределений
- анализ влияния температуры, влажности и скорости воздуха
- сравнение типов охлаждения
- анализ различий по странам, полу и возрасту

- категоризация по количеству рекламаций
- выделение возрастных групп
- расчет доли удовлетворённых пользователей
- проверка соответствия влажности нормативам


### Проверка гипотез

Проверены гипотезы о влиянии:
- типа охлаждения
- пола
- возраста
- количества рекламаций
- страны

Также дополнительно исследованы зависимости между параметрами среды и оценкой комфорта.


### Моделирование

Построена регрессионная модель температуры в помещении с учетом типа охлаждения и других факторов.
